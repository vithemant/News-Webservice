1. Deploy the NEws War in your setup.

2. Start the server and access as :
   a) IP:port/News/front/news   ( Here IP and port is is server IP and port)
    OUTPUT:
    Service will display following things:
    URL, Category, Country, Description, Title
   
   b) IP:port/News/front/news?keyword=example
      OUTPUT:
     i) Service will display follwoing things if news is avialable for searched keyword
      URL, Category, Country, Description, Title
     ii)  Service will display follwoing things if news is not avialable for searched keyword
       ["Nothing found for your Keyword example"]
